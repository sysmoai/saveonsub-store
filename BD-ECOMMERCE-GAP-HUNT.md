# BD E-commerce Completeness — Gap Hunt (consumer-behavior lens)

> What top Bangladeshi online stores have that SAVEONSUB was missing. Grounded in how BD consumers actually shop.

## BD consumer behavior facts that drive these gaps
- **Facebook/Messenger is the #1 commerce channel** in BD — more than websites. WhatsApp alone is not enough.
- **Order anxiety is high** — buyers constantly ask "where is my order / did it work?" A track/status path reduces abandonment.
- **Students are a massive segment** for AI/streaming tools — they need a dedicated, honest zone.
- **Trust before pay** — buyers want to see support hours, channels, response-time, "do you serve my area?".
- **Festival-driven buying** — Eid, Pohela Boishakh, back-to-school spikes; a store needs an offers hub.
- **Word-of-mouth / referral** is the dominant acquisition channel.

## Gaps found → being built this pass

| # | Missing | BD behavior it serves | Fix |
|---|---|---|---|
| 1 | No dedicated **Contact/Support** page | "Are they real? When will they reply? Do they serve me?" | `contact.html` — channels, support hours, response promise, all-BD coverage, ContactPoint schema |
| 2 | No **Track Order** path | Order anxiety — "where is my order?" | `track.html` — enter order ID → one-tap WhatsApp status check |
| 3 | No **Student Zone** | Huge BD student segment | `students.html` — free tools first, student deals, honest verification guidance |
| 4 | No **HTML sitemap** | Human navigation + SEO internal-link equity | `sitemap.html` — every page linked, crawlable |
| 5 | No **Offers/festival hub** | Eid/Boishakh/back-to-school buying | `offers.html` — honest standing offers (bulk/referral/student), festival-ready, NO fake urgency |
